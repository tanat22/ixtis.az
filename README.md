ixtis.az - A University & Faculty Selection Platform for Azerbaijani Students
Welcome to the official repository for the ixtis.az project. This website was created to help prospective students in Azerbaijan with their university and faculty selection process. Our goal is to collect the necessary information in one platform to help students make the right decision about their future.

About the Project
This site is built using simple HTML and is hosted on GitHub Pages. This technology ensures the site's fast and reliable operation. Our main goal is to create a clean, simple, and user-friendly information resource for anyone interested in the university admissions process. You can find information on majors, universities, and admission scores on the site.

How the Site Works
Our site allows users to compare different faculties and universities, learn about admission scores, and research majors that match their potential. The information is based on official sources from the State Examination Center.

Deployment
The website is automatically deployed via GitHub Pages from the main branch of this repository. Any changes pushed to the main branch will be live on ixtis.az within a few minutes.
