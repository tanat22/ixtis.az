We welcome contributions from the community to help improve the ixtis.az website. Whether it's a bug report, a new feature, or a simple typo fix, your help is appreciated.

How to Contribute
Report a Bug: If you find a bug, please open a new issue and describe the problem in detail.

Suggest a Feature: Have an idea for a new feature? Open an issue to discuss it.

Submit a Pull Request:

Fork this repository.

Create a new branch for your feature (git checkout -b feature/my-new-feature).

Make your changes and commit them (git commit -am 'Add new feature').

Push to your branch (git push origin feature/my-new-feature).

Create a new Pull Request from your forked repository to the main branch of this repository.
